FAQ (English)

  • Restore purchases: Open the app → You → Premium → Restore purchases.
  • iCloud sync: iOS Settings → Apple ID → iCloud → LifeScore.
  • Export/Import: In the app → You → Import/Export.
  • Reminders: In the app Settings → Reminders, or iOS Settings → Notifications → LifeScore.
  • Change language: iOS Settings → LifeScore → Language.

Vanliga frågor (Svenska)

  • Återställ köp: Öppna appen → Du → Premium → Återställ köp.
  • iCloud-synk: iOS-inställningar → Apple‑ID → iCloud → LifeScore.
  • Export/Import: I appen → Du → Import/Export.
  • Påminnelser: I appen → Inställningar → Påminnelser, eller iOS-inställningar → Notiser → LifeScore.
  • Byta språk: iOS-inställningar → LifeScore → Språk.
